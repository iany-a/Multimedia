HTML

Requests sent by clients to servers:

GET - req data
POST - send data
PUT/PATCH - modify data
DELETE - erase data

Server response:
response code: 200 = 0K

status codes:
2xx - informational e.g. 101 - switching protocols
2xx - success e.g. 200 - ok, 201 created, 204 no content (used in deletion confirmation as no data was available to be sent back post deletion)
3xx - redirect e.g. 301 - moved permanently, 308 - redirected permanently
4xx - client errors - 400 bad req, 401 unauthorized, 403 forbidden, 404 not found
5xx - server errors e.g. 500 - internal server error, 503 - service unavailable
